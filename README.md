# WebDevProyect4
A map for university newcomers.
Displays where some services like microwave ovens, class help, gym, and medical services are.
Also allows for students to report infrastructural damages like busted bulbs, broken chairs, clogged toilets.

## How to use
The map displays whatever you choose.
We have info on bathrooms, microwave ovens, and academic help.
Also, you may report infrastructural damages, like busted bulbs, or broken chairs.

## Technologies
Meteor
React
Node
Mongo
Git, on Github
Atom as editor
On an Ubuntu OS Thinkpad
Tested on Chrome and Firefox
Lighthouse to check the usability rating
